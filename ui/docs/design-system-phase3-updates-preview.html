Why PageHeader here? The updates page is a first-class inbox feature — it benefits from the same identity-strip pattern used in health.tsx (#1089): a persistent title row that anchors orientation when the user arrives from any nav path. KpiRow surfaces the two key hero numbers (pending / done) without any layout code in the page. MonitorGrid gives the inbox pane (span=4) and detail pane (span=8) a responsive 12-col grid without bespoke CSS.
