Architecture Change

Before

Single cursor.tsx with inline Tabs (Model Config | Settings | Info), internal ScrollArea wrappers, all state inline. No SectionRail. No FormSection anchors.

After

PageShell → CursorForm → StatusSidebar + [SectionRail | FormPane | JsonPane]. FormPane contains FormSection anchors (id=presets|model-mapping|runtime-settings|info) for SectionRail scroll-spy. Manual auth dialog extracted. Helpers in lib/. Pure functions testable in isolation.

SectionRail Note

Used click-to-scroll (scroll-spy) nav. With 4 FormSections in the FormPane scroll container, IntersectionObserver works correctly. Global IntersectionObserver stub added to vitest-setup.ts to prevent test environment errors from SectionRail.
