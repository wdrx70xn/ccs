Identity Strip

The original header had a heavy hero treatment: prominent icon with hover animation, two status stat blocks (Redaction + Retention), and a pill badge. The Monitor archetype calls for a lean identity strip. Migration replaces the bespoke header with:

  • PageHeader with inline ScrollText icon in title (clean, no animation)
  • Redaction status moved to a single Badge in the status slot
  • Retention moved to KpiRow card (more discoverable)
  • Refresh + Health link in actions slot (standard buttons)

Scroll Architecture Note

MonitorLayout wraps children in a ScrollArea, which is correct for pages with discrete card content. The Logs page "Telemetry Stream" tab has a 3-pane layout where each pane owns its own virtualized scroll. Wrapping in MonitorLayout's ScrollArea would create double-scroll nesting and break the fixed-height pane layout.

Resolution: KpiRow sits in a pinned shrink-0 strip. The tab + stream area uses min-h-0 flex-1 to fill remaining viewport. LogsStreamPane takes full height. Each pane (filters, entries, details) has its own scroll via ScrollArea or internal virtualization.
